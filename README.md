# ffmpeg-preview

This repository was designed for making **small FFMpeg builds** that have enough features to\
generate **Image**/**Video** previews and to **extract duration of media files**.

We create executables with [**Github Actions**](https://docs.github.com/en/actions). Version change in the `ffmpeg_tag.txt`\
will trigger build process for **Linux**, **Windows** and **MacOS** systems.
